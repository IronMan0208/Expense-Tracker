# Expense Tracker

A modern Android Expense Tracker App built using Kotlin and Jetpack Compose. This application helps users track their daily expenses, monitor total spending, and manage expenses through a clean and user-friendly interface.

## Features

* Add New Expenses
* Delete Expenses
* Real-Time Total Expense Calculation
* Expense History List
* Modern Material 3 UI
* Simple and User-Friendly Design

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
