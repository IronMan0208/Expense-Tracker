# 💰 Expense Tracker

A modern Android Expense Tracker Application built using **Kotlin**, **Jetpack Compose**, **Room Database**, and **MVVM Architecture**. The app helps users track their daily expenses efficiently while storing data locally for a seamless offline experience.

---

## ✨ Features

* 💰 Add Expenses
* 🗑️ Delete Expenses
* 📊 Real-Time Total Expense Calculation
* 💾 Local Data Persistence with Room Database
* ⚡ Fast and Responsive UI
* 🎨 Modern Material 3 Design
* 🏗️ MVVM Architecture
* 📦 Repository Pattern
* 🔄 Automatic UI Updates using State Management

Run the application on an Android Emulator or Physical Device.
